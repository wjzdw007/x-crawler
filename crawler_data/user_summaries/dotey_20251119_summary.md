**新工具/技术**  
- Gemini 3.0 Pro：支持多模态输入，能直接读取YouTube链接，具备超长上下文和Google搜索集成，适合写作和代码生成，且使用限额较宽松。  
- Codex：日常编程中主要使用，尤其适合写前端代码，常结合Gemini CLI与Claude Code形成更优工作流。  
- Claude Agent Kit：宝玉开源的开发工具包，适合开发Claude Agent SDK，已有用户在优化和应用中反馈良好。

**核心观点/方法论**  
- AI时代推动“即生成即用”交互模式：传统的标准化、可复用、坚固设计原则受限于成本和资源，而AI允许用户即时根据需求生成完全个性化、即用即丢的内容，极大释放创造力与效率。  
- Gemini 3.0不直接替代前端开发，而是开辟新交互方式，类似命令行向图形界面的过渡，促进更丰富和灵活的用户体验与开发可能性。  
- 技术进步不断提升前端和交互体验，前端“死亡”论不成立，新的技术反而带来更多工作量和创新空间。

**实践经验/案例**  
- 使用Gemini 3.0生成的前端演示效果强，但离生产级水平仍有差距，专业前端仍需调整和修复生成代码。  
- 通过提示词设计，Gemini 3能制作复杂交互（如复古拍立得相机网页应用），展示了AI在设计场景化UI上的潜力和创意性。  
- 结合Gemini CLI与Codex、Claude Code构建Coding Agent，是提高开发效率的有效途径。  
- 现有的Claude-agent-kit虽近期无更新，但仍是开发Claude Agent非常实用的参考工具。
