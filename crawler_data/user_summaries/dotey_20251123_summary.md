**新工具/技术**  
- Nano Banana Pro：一款高性能图像生成模型，可实现细节丰富、表达力强的传统水墨与现代题材融合画作风格转换。支持复杂提示词，生成高度一致且艺术效果优异的作品。  
- Claude Agent SDK UI：dotey基于Claude Code Web设计的一套Claude Agent SDK用户界面，功能较为完整，方便结合MCP+Skills打造强大AI聊天客户端。  
- Gemini + Nano Banana Pro结合使用的“教学信息图生成”流程：先用Gemini将文本内容转为结构化信息图文本提示，再通过Nano Banana Pro生成高质量可视化教学信息图。  

**核心观点/方法论**  
- 提示词迭代法：将构思描述给AI，由AI生成初版提示词，经过人类测试验证后反馈调整，反复循环至满意为止。该方法能够高效提升AI生成内容的质量和准确性。  
- 信息图提示词生成分步流程：分解教学设计任务，将内容与设计指令分离，确保信息图内容严谨且设计规范化，提升信息传递效果。  
- 传统文化与现代元素结合的创作思路：用AI将传统中国工笔水墨画手法与现代日常生活、科幻元素相融合，展现跨时代、跨文化的视觉冲击力和趣味性。  

**实践经验/案例**
